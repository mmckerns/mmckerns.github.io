1== Notes on mystic examples ==
2Dependencies:
3 - Several of the examples require matplotlib to be installed.
4 - For the examples that use matplotlib, see trac ticket #36 for more details.
5
6Other dependencies:
7 - Examples with prefix "example" are part of the tutorial (TRY THESE FIRST).
8 - All examples with prefix "example" should run without new dependencies.
9 - All examples with prefix "test_" should run without new dependencies.
10 - All examples with prefix "gplot_" requre gnuplot-py for visualization.
11
12Exceptions to the rule:
13 - The following examples also require scipy to be installed:
14   . test_lorentzian.py
15   . test_mogi_anneal.py,
16
17Special examples:
18 - All examples with prefix "rosetta_" require park to be installed. (tests on version park-1.2).  Run with "--park" to execute with park. See "--help" for more options.
19

23Notes on the "ffit" tests/examples:
24 - test_ffit: The fitting problem whose exact solution is 8th order Chebyshev polynomial of the first kind.  This example uses a Ctrl-C signal handler. Try ctrl-c as the differential_evolution strategy is running.
25 - test_ffit2: The fitting problem whose exact solution is 16th order Chebyshev polynomial of the first kind.  Also uses the signal_handler.
26 - test_ffitB: Same as test_ffit.py, but uses DifferentialEvolutionSolver2 instead of DifferentialEvolutionSolver.
27 - test_ffitC: Same as test_ffit.py, but uses scipy_optimize.fmin.
28 - test_ffitD: Same as test_ffit.py, but uses scipy_optimize.diffev.
29
30
31Notes on the "mogi" tests/examples:
32 - test_mogi.py: One mogi source with noise, comparison between DE and Conjugate Gradient, Simplex, and least squares (Levenberg Marquardt). CG / lsq don't work very well. lsq should work when bounds on the parameters are given, but minpack (wrapped by scipy) version doesn't seem to support bounds.
33 - test_mogi_anneal.py: tests with scipy simulated annealing, but hasn't been tuned, so again, doesn't work at all.
34 - test_mogi2.py: two mogi sources
35 - test_mogi3.py: reimplements test_mogi
