12Functions
13=========
14
15Mystic provides a set of standard fitting functions that derive from
16the function API found in `mystic.models.abstract_model`. These standard
17functions are provided::
18    sphere     -- De Jong's spherical function
19    rosen      -- Rosenbrock's function
20    step       -- De Jong's step function
21    quartic    -- De Jong's quartic function
22    shekel     -- Shekel's function
23    corana     -- Corana's function
24    fosc3d     -- Trott's fOsc3D function
25    nmin51     -- Champion's NMinimize test function
26    griewangk  -- Griewangk's function
27    zimmermann -- Zimmermann's function
28    peaks      -- NAG's peaks function
29    venkat91   -- Venkataraman's sinc function
30    schwefel   -- Schwefel's function
31    ellipsoid  -- Pohlheim's rotated hyper-ellipsoid function
32    rastrigin  -- Rastrigin's function
33    powers     -- Pohlheim's sum of different powers function
34    ackley     -- Ackley's path function
35    michal     -- Michalewicz's function
36    branins    -- Branins's rcos function
37    easom      -- Easom's function
38    goldstein  -- Goldstein-Price's function
39    paviani    -- Paviani's function
40    wavy1      -- a simple sine-based multi-minima function
41    wavy2      -- another simple sine-based multi-minima function
42
43
44Models
45======
46
47Mystic also provides a set of example models that derive from the model API
48found in `mystic.models.abstract_model`. These standard models are provided::
49    poly       -- 1d model representation for polynomials
50    circle     -- 2d array representation of a circle
51    lorentzian -- Lorentzian peak model
52    br8        -- Bevington & Robinson's model of dual exponential decay
53    mogi       -- Mogi's model of surface displacements from a point spherical
54                  source in an elastic half space
55
56Additionally, circle has been extended to provide three additional models,
57each with different packing densities::
58    - dense_circle, sparse_circle, and minimal_circle
59
60Poly also provides additional models for 2nd, 4th, 6th, 8th, and 16th order
61Chebyshev polynomials::
62    - chebyshev2, chebyshev4, chebyshev6, chebyshev8, chebyshev16
63
64
