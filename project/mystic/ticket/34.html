Description (last modified by mmckerns) (diff)

Mystic's tutorials are bundled in the mystic-0.1a1 release, but are not installed. This is to avoid a matplotlib dependency. Maybe a nice addition would be to provide a mystic-examples.tgz, with all the basic examples.

comment:3 Changed 6 years ago by mmckerns

It's not desirable to have the tutorials installed as part of the package -- however, it is desirable for them to be available as sources. Thus the current approach is good when installing from source. However, those who do an egg install, will have to get the examples separately. Making a simple .tgz of the "mystic.examples" directory (or alternately just the tutorial examples), and posting it to the release page is probably the best solution.
